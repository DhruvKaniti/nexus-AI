# Nexus AI

> An AI-powered global intelligence platform for monitoring, investigating, and understanding world events.

---

## Overview

Nexus AI brings together global events into a single interactive dashboard. Instead of searching through multiple news outlets and information sources, users can explore ongoing situations on a live world map, investigate events with AI, and monitor intelligence updates in one place.

The goal of Nexus AI is to make global information easier to understand by combining visualization, automation, and AI-powered analysis.

---

## Features

### Interactive Global Map
Visualize events around the world through an interactive map with dynamic markers.

### AI Investigation
Investigate an event and receive contextual analysis generated by AI.

### Live Intelligence Feed
Browse continuously updated event summaries and key information.

### Modern Dashboard
A mission-control inspired interface designed for quickly understanding developing situations.

## Why We Built It

Global events generate an overwhelming amount of information every day. We wanted to build a platform that helps users quickly understand important situations without having to manually gather information from multiple places.

Nexus AI combines visualization and AI into a single workflow that makes exploring global events faster and more intuitive.

---

## How We Used Codex & GPT-5.6

Codex and GPT-5.6 were used throughout development as engineering assistants rather than code generators.

They helped us:

- Debug backend API issues
- Improve frontend components and layouts
- Troubleshoot deployment on Render and Vercel
- Refactor code during rapid iteration
- Evaluate implementation approaches
- Improve documentation and project organization

Using these tools allowed us to spend more time building features and refining the user experience while reducing time spent on repetitive debugging.
